To predict the amount of bikes for the next hour, run the train.py file (python train.py) in the terminal,
then run the predict file (python predict.py) in the terminal.

An additional pipeline (pipeline2.py) is provided in the zip file, and is used for some visualisations in the exploration.ipynb file.
This pipeline is not used for anything else, the final pipeline is a function in the train.py script.
